# Juego del Gusanito (Scratch)

Este es mi primer proyecto en GitHub.  
El juego está hecho en Scratch y trata de un gusanito que come manzanas y crece.

## ¿Cómo se juega?
- Usa las flechas para mover al gusanito.
- El gusanito crece cada vez que come una manzana.

## Cómo está hecho
Está programado en **Scratch**, usando bloques de movimiento, detección y clonación.
